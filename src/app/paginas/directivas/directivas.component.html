Directivas

Hay 3 tipos de directivas en Angular:

Components:
Son directivas con una plantilla de html
Estructurales:
Cambian el DOM añdiendo o removiendo elementos, por jemplo *ngIf
Attribute directive:
Cambian la apariencia de un elemento del DOM, por ejemplo NgStyle

Podemos crear nuestras propias directivas usando el comando ng generate directive $nombre_de_nuestra_directiva
